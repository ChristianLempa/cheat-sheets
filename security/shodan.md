# Shodan

## Filter IP range

```
net:<ip range>
```

## Filter port

```
port:<port>
```

## Filter location

```
city:"<city>" country:<country_code>
geo:<coords>
```

## Filter hostname

```
hostname:<hostname>
```

## Filter operating system

```
os:<operating system>
```

## Filter dates
### Acceptable formats are: day/month/year or day-month-year

```
before:<date>
after:<date>
```
