# Environment Variables in Windows
## How to use them in PowerShell
Environment Variables can be used in [[powershell|PowerShell]] with the prefix `$env:`.

**Example***:
*Variable:*  `%APPDATA%`
*In Powershell:* `$env:APPDATA`
